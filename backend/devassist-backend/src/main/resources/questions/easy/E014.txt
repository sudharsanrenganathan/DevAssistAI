ID: E014
TITLE: Count Digits in a Number
DIFFICULTY: EASY

DESCRIPTION:
Given an integer, count the number of digits in it.

INPUT FORMAT:
- One line containing an integer

OUTPUT FORMAT:
- Print the number of digits

SAMPLE TEST CASES:
INPUT:
123
OUTPUT:
3
EXPLANATION:
123 has three digits.

INPUT:
5
OUTPUT:
1
EXPLANATION:
Single digit number.

INPUT:
1000
OUTPUT:
4
EXPLANATION:
1000 has four digits.

HIDDEN TEST CASES:
INPUT:
0
OUTPUT:
1

INPUT:
987654
OUTPUT:
6

CONSTRAINTS:
- 0 ≤ number ≤ 10^18
