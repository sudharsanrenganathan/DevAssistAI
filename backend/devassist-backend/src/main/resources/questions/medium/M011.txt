ID: M011
TITLE: Remove Duplicates from Array
DIFFICULTY: MEDIUM

DESCRIPTION:
Given a sorted array, remove duplicates and print unique elements.

INPUT FORMAT:
- First line contains integer N
- Second line contains N sorted integers

OUTPUT FORMAT:
- Print unique elements in order

SAMPLE TEST CASES:
INPUT:
5
1 1 2 2 3
OUTPUT:
1 2 3
EXPLANATION:
Duplicates removed.

INPUT:
3
1 1 1
OUTPUT:
1
EXPLANATION:
Only one unique element.

INPUT:
4
1 2 3 4
OUTPUT:
1 2 3 4
EXPLANATION:
No duplicates.

HIDDEN TEST CASES:
INPUT:
6
0 0 1 1 2 2
OUTPUT:
0 1 2

INPUT:
2
5 5
OUTPUT:
5

CONSTRAINTS:
- 1 ≤ N ≤ 10^5
