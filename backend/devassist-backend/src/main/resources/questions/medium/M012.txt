ID: M012
TITLE: Merge Two Sorted Arrays
DIFFICULTY: MEDIUM

DESCRIPTION:
Given two sorted arrays, merge them into one sorted array.

INPUT FORMAT:
- First line contains N
- Second line contains N integers
- Third line contains M
- Fourth line contains M integers

OUTPUT FORMAT:
- Print merged sorted array

SAMPLE TEST CASES:
INPUT:
3
1 3 5
3
2 4 6
OUTPUT:
1 2 3 4 5 6
EXPLANATION:
Arrays merged and sorted.

INPUT:
1
1
1
2
OUTPUT:
1 2
EXPLANATION:
Single elements merged.

INPUT:
2
1 2
0

OUTPUT:
1 2
EXPLANATION:
Second array empty.

HIDDEN TEST CASES:
INPUT:
3
1 2 3
3
4 5 6
OUTPUT:
1 2 3 4 5 6

INPUT:
0

2
7 8
OUTPUT:
7 8

CONSTRAINTS:
- 0 ≤ N, M ≤ 10^5
