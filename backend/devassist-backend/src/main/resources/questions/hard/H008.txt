ID: H008
TITLE: Dijkstra Shortest Path
DIFFICULTY: HARD

DESCRIPTION:
Given a weighted graph, find shortest path from source to all vertices.

INPUT FORMAT:
- First line contains V and E
- Next E lines contain u v w
- Last line contains source

OUTPUT FORMAT:
- Print shortest distances from source

SAMPLE TEST CASES:
INPUT:
3 3
0 1 4
0 2 1
2 1 2
0
OUTPUT:
0 3 1
EXPLANATION:
Shortest distances printed.

INPUT:
2 1
0 1 5
0
OUTPUT:
0 5
EXPLANATION:
Single edge.

INPUT:
1 0
0
OUTPUT:
0
EXPLANATION:
Single node.

HIDDEN TEST CASES:
INPUT:
4 4
0 1 1
1 2 2
2 3 3
0 3 10
0
OUTPUT:
0 1 3 6

INPUT:
3 1
0 1 7
0
OUTPUT:
0 7 INF

CONSTRAINTS:
- 1 ≤ V ≤ 10^5
