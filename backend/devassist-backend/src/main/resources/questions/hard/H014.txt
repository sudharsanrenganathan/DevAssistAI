ID: H014
TITLE: Serialize and Deserialize Binary Tree
DIFFICULTY: HARD

DESCRIPTION:
Design an algorithm to serialize and deserialize a binary tree.

INPUT FORMAT:
- Level order traversal with null as -1

OUTPUT FORMAT:
- Print serialized string
- Print tree again in level order

SAMPLE TEST CASES:
INPUT:
1 2 3 -1 -1 4 5
OUTPUT:
1,2,3,-1,-1,4,5
1 2 3 -1 -1 4 5
EXPLANATION:
Tree is preserved.

INPUT:
-1
OUTPUT:
-1
-1
EXPLANATION:
Empty tree.

INPUT:
1
OUTPUT:
1
1
EXPLANATION:
Single node.

HIDDEN TEST CASES:
INPUT:
2 1 -1
OUTPUT:
2,1,-1
2 1 -1

INPUT:
3 -1 -1
OUTPUT:
3,-1,-1
3 -1 -1

CONSTRAINTS:
- Number of nodes ≤ 10^5
