ID: H004
TITLE: Trapping Rain Water
DIFFICULTY: HARD

DESCRIPTION:
Given heights of bars, find how much rain water can be trapped.

INPUT FORMAT:
- First line contains N
- Second line contains N integers

OUTPUT FORMAT:
- Print total trapped water

SAMPLE TEST CASES:
INPUT:
12
0 1 0 2 1 0 1 3 2 1 2 1
OUTPUT:
6
EXPLANATION:
Total trapped water is 6.

INPUT:
3
2 0 2
OUTPUT:
2
EXPLANATION:
2 units trapped.

INPUT:
3
1 2 3
OUTPUT:
0
EXPLANATION:
No trapping.

HIDDEN TEST CASES:
INPUT:
5
3 0 0 2 0
OUTPUT:
4

INPUT:
4
4 2 0 3
OUTPUT:
4

CONSTRAINTS:
- 1 ≤ N ≤ 10^5
