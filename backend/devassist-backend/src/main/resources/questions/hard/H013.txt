ID: H013
TITLE: Median of Two Sorted Arrays
DIFFICULTY: HARD

DESCRIPTION:
Given two sorted arrays, find the median of the combined array.

INPUT FORMAT:
- First line contains N
- Second line contains N integers
- Third line contains M
- Fourth line contains M integers

OUTPUT FORMAT:
- Print the median

SAMPLE TEST CASES:
INPUT:
2
1 3
1
2
OUTPUT:
2
EXPLANATION:
Combined array is [1,2,3].

INPUT:
2
1 2
2
3 4
OUTPUT:
2.5
EXPLANATION:
Median is average of 2 and 3.

INPUT:
1
1
0

OUTPUT:
1
EXPLANATION:
Only one array.

HIDDEN TEST CASES:
INPUT:
3
1 2 3
3
4 5 6
OUTPUT:
3.5

INPUT:
0

1
7
OUTPUT:
7

CONSTRAINTS:
- 0 ≤ N, M ≤ 10^5
